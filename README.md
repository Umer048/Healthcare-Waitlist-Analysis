# 🏥 Healthcare Waitlist Analysis Dashboard
**End-to-End Data Analytics Project** | *By Muqaddas Fatima*

---

## 📌 Project Overview
Identifying bottlenecks in hospital patient flow using Power BI. This project transforms complex healthcare data into actionable insights to improve patient care and resource allocation.

## 🛠️ Technical Process
* **Data Cleaning:** Resolved "Model object name" and "Date Locale" errors in **Power Query**.
* **Modeling:** Merged Inpatient and Outpatient datasets into a unified model.
* **DAX Logic:** Developed dynamic measures to switch between **Average** and **Median** wait times.

## 📊 Key Insights
* **Latest Month Waitlist:** 629K
* **Year-over-Year Increase:** +11.7%
* **Critical Departments:** Orthopaedics, ENT, and Gynaecology identified as high-demand areas.
